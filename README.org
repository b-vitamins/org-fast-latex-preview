#+title: org-fast-latex-preview

Fast, asynchronous LaTeX previews for Org buffers.

* What Is OFLP?

=org-fast-latex-preview= is an in-buffer LaTeX preview package for Org.

It provides:

- batched asynchronous rendering
- persistent SVG cache
- quiet failure reporting
- reveal-at-point and refresh-on-edit behavior
- integration with Org's existing preview configuration

It does not handle export.  It does not replace Org generally.  It only
improves LaTeX previews inside Org buffers.

* Requirements

- Emacs 29.1+
- Org 9.7+
- =latex=
- =dvisvgm=
- LaTeX =preview= package

Optional:

- =xelatex= or =dvilualatex=, if your Org preview setup uses them

* Installation

Add the package to =load-path= and require it:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/org-fast-latex-preview/lisp")
(require 'org-fast-latex-preview)
#+end_src

Enable it in Org buffers:

#+begin_src emacs-lisp
(add-hook 'org-mode-hook #'org-fast-latex-preview-mode)
#+end_src

Or enable it globally:

#+begin_src emacs-lisp
(global-org-fast-latex-preview-mode 1)
#+end_src

That is enough for normal use.  OFLP will use Org's existing preview state by
default.

* Basic Use

With =org-fast-latex-preview-mode= enabled:

- =org-latex-preview= uses OFLP in that buffer
- =org-clear-latex-preview= uses OFLP in that buffer
- edits queue a refresh after an idle delay
- moving point over a preview reveals the source

Main commands:

- =org-fast-latex-preview=
- =org-fast-latex-preview-buffer=
- =org-fast-latex-preview-region=
- =org-fast-latex-preview-subtree=
- =org-fast-latex-preview-refresh=
- =org-fast-latex-preview-clear=
- =org-fast-latex-preview-clear-cache=

* Configuration

Most users should configure Org normally and leave OFLP on its default profile:

#+begin_src emacs-lisp
(setq org-fast-latex-preview-profile 'auto)
#+end_src

=auto= follows native Org preview state.  If you already customize:

- =org-format-latex-header=
- =org-latex-packages-alist=
- =org-preview-latex-default-process=
- =org-format-latex-options=

OFLP will reuse that information.

Useful OFLP-specific options:

- =org-fast-latex-preview-cache-directory=
- =org-fast-latex-preview-max-concurrent-jobs=
- =org-fast-latex-preview-failure-strategy=
- =org-fast-latex-preview-max-image-size=
- =org-fast-latex-preview-override-org-commands=

If you want OFLP to follow Org's preview/export setup more strictly:

#+begin_src emacs-lisp
(setq org-fast-latex-preview-profile 'org-state)
#+end_src

If you want to force a specific compiler/preamble path:

#+begin_src emacs-lisp
(setq org-fast-latex-preview-profile 'manual)
#+end_src

Additional preview-only macros or packages can be added with:

#+begin_src emacs-lisp
(setq org-fast-latex-preview-extra-preamble
      "\\usepackage{mathtools}\n\\newcommand{\\dd}{\\mathop{}\\!\\mathrm{d}}\n")
#+end_src

* Limitations

- graphical Emacs only
- SVG previews via =dvisvgm= only
- no export integration
- no terminal image preview support
